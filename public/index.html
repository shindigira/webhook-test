üìã Overview

This project provides a webhook endpoint for receiving and processing Instagram webhook events. It's built with Express.js and deployed on Railway. The dashboard below shows real-time status and incoming webhook events.

üöÄ Endpoints

GET /api/webhook

Webhook verification endpoint for Instagram subscription setup.

Query Parameters:

  • hub.mode - Must be "subscribe"
  • hub.verify_token - Your verification token
  • hub.challenge - Challenge string from Instagram
POST /api/webhook

Receives webhook events from Instagram.

All incoming events are logged to the console.

GET /api/health

Health check endpoint. Returns "ok" if the server is running.

üìù Setup Instructions

  1. Deploy this project to Railway
  2. Set the VERIFY_TOKEN environment variable in Railway
  3. Configure your Instagram app's webhook URL to point to https://your-domain.railway.app/api/webhook
  4. Use the same VERIFY_TOKEN in your Instagram app settings
  5. Subscribe to the webhook events you want to receive
  6. Monitor incoming events in real-time on this dashboard
